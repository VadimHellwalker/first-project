Учебный проект Яндекс.Практикум по курсу Data Science плюс "Обучение с учителем: качество модели".

Описание исследования
В этом исследовании предстоит выступить в роли стажёра в отделе цифровых технологий интернет-магазина «В один клик». Магазин продаёт разные товары: для детей, для дома, мелкую бытовую технику, косметику и даже продукты. Отчёт магазина за прошлый период показал, что активность покупателей начала снижаться. В работе проведён анализ данных по результатам которого разработано предложение по взаимодействию с клиентами магазина.

Цель исследования
Разработать решение, которое позволит персонализировать предложения постоянным клиентам, чтобы увеличить их покупательскую активность.

Задачи исследования
Построить модель, которая предскажет вероятность снижения покупательской активности клиента в следующие три месяца.
Включить дополнительные данные финансового департамента о прибыльности клиента за последние три месяца.
Выделить сегмент покупателей и разработать для него персонализированные предложения.

Результаты
- В результате решения задачи исследования построена модель, которая предсказала вероятность снижения покупательской активности клиента в следующие три месяца.
- Включены дополнительные данные финансового департамента о прибыльности клиента за последние три месяца.
- Выделен сегмент покупателей для которого разработаны персонализированные предложения.
- Исходные данные представлены четырьмя датасетами:

1. market_file.csv содержит данные о поведении покупателя на сайте, о коммуникациях с покупателем и его продуктовом поведении.
2. market_money.csv содержит данные о выручке, которую получает магазин с покупателя, то есть сколько покупатель всего потратил за период взаимодействия с сайтом.
3. market_time.csv — это данные о времени (в минутах), которое покупатель провёл на сайте в течение периода.
4. money.csv — это данные о среднемесячной прибыли покупателя за последние 3 месяца: какую прибыль получает магазин от продаж каждому покупателю.

- Пропусков и явных дубликатов в данных не было. Исправлены опечатки.
- Для поиска лучшей модели был собран пайплайн, в котором было проведено кодирование двумя кодировщиками и масштабирование двумя скейлерами признаков по итогу которого с помощью дерева решений с учётом заданных гиперпараметров была отобрана одна из четырёх моделей, показавшая лучшую метрику. Среди рассмотренных моделей были логистическая регрессия, метод опорных векторов, метод k-ближайших соседей и дерево решений.
- Лучшей моделью оказался метод опорных векторов с параметром регуляризации C=4.
- Анализ группы с высокой вероятностью снижения покупательской активности и наиболее высокой прибыльностью показал, что нужно детально разобраться как устроена акционная политика и почему частое приобретение товаров по акции снижает покупательскую активность. Кроме этого нужно проанализировать преимущества преимум сегмента, похоже их недостаточно для стимулирования совершения покупок. Также нужно изучить ассортимент кухонной посуды и понять почему предпочитающие эту категорию имеют риск к снижению покупательской активности. И нужно проверить клиентов которые накапливают большое количество товаров в корзине, но покупку не совершают. Возможно отпугивает цена и можно рекомендовать раздавать купоны на скидку.
